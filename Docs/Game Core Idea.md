# 当前游戏的背景

## 期望的游戏流程
- 启动游戏, 玩家看到的是unity和公司logo, 此时游戏正在初始化
- 进入游戏主菜单, 玩家能看到主菜单
- 游戏在主菜单点击开始游戏按钮后 splash 进入关卡
- 关卡中通关或者退出到主菜单则 splash 到主菜单
